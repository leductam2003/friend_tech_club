# SETTINGS
设置钱包
把你的ft私钥放在 wallets.txt 里面就行

自己选gas
在config.ini把gas_auto改成no 然后修改gas_price

自动gas
在config.ini把gas_auto改成yes gas_price不用管，不填也行
